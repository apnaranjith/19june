Tags

Tags are of 2 kinds: 
They are-

Semantic Tags

These are the tags that define the meaning of the content residing within it to the browser and the user.

These tags include:
  1. <header> tag
  2. <aside> tag
  3. <section >tag
  4. <article >tag
  5. <footer>tag
  6. <form>tag
  7. <table>tag
  8. <details>tag
  9. <mark>tag
  10. <main>tag
  11. <figure>tag
  12. <figcaption>tag
  13. <summary>tag
  14. <time>tag

Non-Semantic Tags

These are the tags that don't define the meaning of the content residing within it to the browser and the user.

These tags include:
  • <Div > tag
  • <Span > tag
